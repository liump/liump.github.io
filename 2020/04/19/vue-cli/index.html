安装 vue-cli 前需要安装 node.js

# vue-cli2

  1. 安装: 可以使用 npm install -g vue-cli 或者 cnpm install -g vue-cli

  2. 使用: vue init webpack projectName (projectName 项目名字)

  3. 配置: vue-router 官方路由选 Y

  4. ESLint: 代码风格管理器,视情况选择

  5. 启动:

    • cd projectName
    • npm run dev
    • 打开浏览器 localhost:8080 查看项目是否正常启动

  6. router 实现页面跳转

    • 在 html 结构中使用 <router-link to=''></router-link> to 指向的是要跳转的位置

  7. router 配置

    • router/index.js 配置路由
    • index.js 里的 routes 添加路由对象

  8. 嵌套路由

    • router/index.js 配置路由
    	{
		path: '/a',
		components: a,
		children: [
			{
				path: '/a1',
				components: a1
			}
		]
	}
    • 挂载子路由: a 组件中添加 router-view

  9. hash 与 history 模式 ::: tip

  10. 地址栏中,经常出现 # ,是什么作用,能不能去掉# ?

  11. hash 模式: 地址栏包含#符号,#以后的不被后台获取

  12. history 模式: 具有对URL历史记录修改的功能

  13. 在微信支付/分享URL作为参数传递时,#是特殊字符,很多场合不能满足需求

  14. history 需要后台配合,处理404的问题 :::

router/index.js 中添加 mode: 'history'

  1. 插件的安装与卸载
    • 项目中安装
      • npm install eslint --dev
    • 项目中卸载
      • npm uninstall eslint --dev

# vue-cli3

  1. 安装 vue-cli3: npm install -g @vue/cli
  2. vue-cli3 的图形界面: vue ui
    • 可以使用 vue ui 进行项目管理
    • 图形界面的问题,安装很慢
  3. 手动创建项目
    • vue create projectName
      • 配置选择:
        • 默认 babel/ESLint
        • 手动选择安装 router 等插件
    • cd projectName
    • npm run serve
  4. 插件配置
    • 使用 vue ui 配置插件
    • 使用 vue add eslint 配置插件
