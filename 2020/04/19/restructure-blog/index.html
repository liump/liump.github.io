# Vuepress

Vue 驱动的静态网站生成器,可以用于文档生成、blog等.

@vuepress/theme-blog

# 插件安装

# flowchart

流程图 flowchart 插件.

  1. 确保项目中有 vuepress 而不是全局安装
  2. 安装 flowchart 插件 yarn add vuepress-plugin-flowchart -D
  3. 配置:
    // .vuepress/config.js
module.exports = {
  plugins: [
    'flowchart'
  ]
}
  4. 语法:
    @flowstart [preset]

<!-- Your flowchart code here. -->

@flowend

// exam
@flowstart
st=>start: Start
e=>end: End

st->e
@flowend

# mathjax

数学公式 vuepress-plugin-mathjax 插件

  1. 安装

    npm install -D vuepress-plugin-mathjax

  2. 使用

    // .vuepress/config.js
module.exports = {
  plugins: [
    [
      'vuepress-plugin-mathjax',
      {
        target: 'svg',
        macros: {
          '*': '\\times',
        },
      },
    ],
  ],
}
