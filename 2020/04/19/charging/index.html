# 字段

设备实体字段

字段名 数据类型 说明 备注
id String 主键ID
orgId String 机构
serialNumber String 设备编号
iMei String IMEI
internetNumber String 物联网卡号
leagueScale BigDecimal 加盟商比例
operatorScale BigDecimal 运营商比例
annualFee BigDecimal 年费
moduleState Integer 模块状态 1-离线，2-在线
equipmentState Integer 设备状态 0-已删除，1-离线，2-在线
netSignal String 信号
netModel String 网络模式
isUse Integer 是否可用 1-不可用，2-可用
equipmentType Integer 设备类型 0-充电桩（投币式），1-充电站（功率），2-充电站（手机选路）
equipmentAddr String 设备地址:关联地址表
longitude String 经度
latitude String 纬度
dueDate String 到期日期
remark String 备注
createDate String 创建时间
createUser String 创建人ID
updateDate String 创建时间
updateUser String 创建人ID

续费表字段

字段名 数据类型 说明 备注
id String 主键ID
serialNumber String 设备编号
orgId String 机构
renewNumber String 续费编号
renewAmount BigDecimal 续费金额
paymentTime String 付款时间
paymentState Integer 付款状态 1-未支付，2-已支付
paymentWay Integer 付款方式 0-微信支付……
createDate String 创建时间
createUser String 创建人ID
updateDate String 创建时间
updateUser String 创建人ID

用户表（user）

类型 字段名 说明
String id 用户id
String cname 用户名
String pwd 用户登录密码
String trueName 用户真实姓名
String visible 0，不需要显示（已被删除）1.需要显示
boolean ifLock 1-正常使用，0-停用状态(不可登录)（此字段控制是否允许登录）
String gender 性别 1-男，2-女
String email 邮箱
String address 用户地址
String orgId 所属机构
String position 用户职位
String phone 手机号
String telephoneNum 座机号
String bornDate 出生日期
String changePwd 首次登录后 0-不需要，1-需要
String lastLoginDate 最后登录日期
String lastLoginIp 最后登录IP
String lastLoginDesc 最后登录描述
String idcardNo 身份证号
boolean ifAdmin 0,否 1，是
String pwdTime 设置密码的时间

角色表（role）

类型 字段名 说明
String id 角色id
String name 角色名称
String orgId orgID
String remark 备注
String roleCode 角色代码

# 9# 登录 [ login ]

    • 移动端传roleId：用户选择登陆的角色id

  pwd: '',
  rememberMe: false,
  isPC: true,
  roleId
}

  • Response 200

    {
  flag:1
}
