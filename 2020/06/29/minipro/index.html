# 小程序开发

简介: 为了拓展前端技能,准备小程序的开发,开发环境使用uni-app + uniCloud 实现,api方面使用聚合数据.

从6月1日开始到6月29日过审核,断断续续开发了一个月.

# 准备过程:

  1. 下载HbuildX.
  2. 通过HbuildX创建uni-app项目,勾选uniCloud云开发.
  3. 配置文件 manifest.json 添加 AppID.
  4. uniCloud 云开发环境目前是免费状态,选用的阿里云.

# 技术栈

vue + uni-app + uniCloud + uview + lodash + moment

# 构思

  1. 我要做一个什么样的小程序?
  • 我的想法是做一个有 tarbar 的小程序.
  • 分为: 首页/发现页/blog
  • blog 原本的想法是 web-view 链接 github的pages也就是 liump.github.io,但个人认证的小程序没有 web-view 的使用功能,只有企业的小程序才可以使用 web-view.
  • 小程序改为 首页 / 发现页
  1. 首页
  • 参考了 one，好看.
  • 左上角是日期
  • 右上角是天气
  • 正中间是 swiper,内容是图片加上励志话语
  1. 发现页
  • 发现页分为 新闻/笑话/历史上的今天
  • 新闻页在本地开发完了才知道个人小程序并不能使用 web-view.
  • 历史上的今天在审核的时候才知道这个内容涉及政治,个人小程序也不能使用,只能企业小程序才能.
  • 笑话使用聚合的api,但是我发现一点也不好笑...
  • 于是发现页改为 book.图书的分类、查找与简介.

# 搭建开发环境

  1. 使用 HbuildX 构建 uni-app + uniCloud 项目.
  2. 下载 lodash moment 工具库.
  3. 下载 uview UI库.
  4. 引入工具库及UI库.
// 引入 moment 日期库
import moment from './common/js/moment.js';
// 原型挂载 moment 库,使用: this.$moment().format()
Object.defineProperty(Vue.prototype, '$moment', {
  value: moment
});
  • js 工具库 使用 Object.defineProperty 挂载.
  1. 申请聚合免费数据,部分api免费但每天100次.注意:申请api需要个人实名认证,上传身份证的那种.

# 收获

记录此次小程序开发收获:

  1. lodash 和 moment 的使用.
  2. severless 云开发经验.
  3. uview UI库,多了一个UI层库的使用.
  4. 了解了个人不能使用 web-view .
  5. 小程序 tarbar 需要通过 pages.json 实现.
  6. js 工具库通过 Object.defineProperty 挂载.
  7. serveless 云数据库、云存储及云函数的使用.
