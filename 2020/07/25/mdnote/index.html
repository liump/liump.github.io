# 目标

  1. 该笔记本应用允许用户以 Markdown标记语言来写笔记
  2. 支持 Markdown的实时预览
  3. 用户可以添加任意多条笔记
  4. 笔记可以在用户下次打开应用时重新加载出来

# marked 第三方库

可以将 Markdown 文本转义成HTML结构

# moment.js 日期处理库

# 笔记数据结构

  • id: 笔记的唯一标识符
  • title: 笔记的标题
  • content: 笔记 Markdown 文本
  • created: 笔记创建日期
  • favorite: 是否收藏笔记

# 知识点

  1. 计算属性的值基于它的英爱进行缓存,如果没有必要是不会重新运行函数,防止无用的计算.
  2. 文本插值处于安全考虑不会渲染 HTML 可以使用 v-html 指令,不建议把用户提供的 HTML 直接渲染,防止恶意代码攻击(XSS攻击).
  3. created：Vue 设置好数据属性及其初始值。
  4. v-on 指令：值为函数，该事件触发调用
  5. 动态 CSS类： <div :class="{actived: note === selected}"></div>。
  6. 布尔值反转：bool ^= true。
  7. filter 过滤器：Vue.filter('date', time => mement(item).format('DD/MM/YY, HH:mm'))，format 方法内为日期格式。
  8. 本地存储
  • localStorage.setItem(key, value) 存数据，把 value 存储到 key 中。
  • localStorage.getItem(key) 取 key 对应数据。
