# 组件和Props

函数式组件(无状态组件)
React.Component

<script type="text/babel">
// 函数式组件 - 无状态
function Hello(props) {
  return <div>
    <h1>Hello,{props.name}</h1>
  </div>
}

// React.Component 组件 - 有状态,this指向React.Component实例
class Hello extends React.Component{
  render(){
    return <div>
      <h1>Hello,{this.props.name}</h1>
    </div>
  }
}

// 渲染
ReactDom.render(
  <Hello name="liump">,
  document.getElementById('app')
)
</script>

# 关于业务函数的this问题

不能以es6的方式直接写业务函数,如: updateData(){} ,这样会导致作用域异常.

解决方法:

  1. 直接使用 updateData = () => {} ,使用箭头函数改变作用域
  2. 在 constructor 中手动改变指向, 如: this.updateData = this.updateData.bind(this)
  3. 在事件上处理 onClick = {() => this.updateData}
  4. 在事件上处理的另一种方式 onClick = {this.updateData.bind(this)}

# 列表渲染

<script type="text/babel">
class List extends React.Component{
  state = {
    list: [1,2,3,4,5]
  }
  render(){
    const arr = this.state.list;
    const listItem = []
    arr.map((item) => {
      let li = <li>{item}</li>
      listItem.push(li)
    })
    return <div>
      <ul>
        {listItem}
      </ul>
    </div>
  }
}
</script>
