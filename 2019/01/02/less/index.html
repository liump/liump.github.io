# 变量

``` less
@test_width:300px;

.box{
    width: @test_width;
    height: @test_width;
    background-color:yellow;
}
```

# 混合

``` less
    @test_width:300px;

    .box{
        width: @test_width;
        height: @test_width;
        background-color:yellow;

        .border;
    }

    .border{
        border: 1px solid pink;
    }
```

# 带参数的混合

``` less
.border_02(@border_width){
    border: @border_width solid yellow;
}
.test_hunhe{
    .border_02(30px);
}
```

# 带默认值的混合

``` less
.border_03(@border_width:10px){
    border: @border_width solid green;
}

.test_hunhe_03{
    .border_03();
}
```

# 匹配模式(相当于 JS 的流程控制)

``` less
/*
.sanjiao{
    width: 0;
    height: 0;
    overflow: hidden;

    border-width: 10px;
    border-color:  transparent transparent red transparent;
    border-style: dashed dashed solid dashed;
}
*/
.triangle(top,@w:5px,@c:#ccc){
    border-width: @w;
    border-color: transparent transparent @c transparent;
    border-style: dashed dashed solid dashed;
}
.triangle(bottom,@w:5px,@c:#ccc){
    border-width: @w;
    border-color: @c transparent transparent  transparent;
    border-style: solid dashed dashed  dashed;
}
.triangle(@_,@w:5px,@c:#ccc){
    # @_ 表示不管匹配到哪一个始终带着这个,注意其他参数必须带上
    width: 0;
    height: 0;
    overflow: hidden;
}

.sanjiao{
    .triangle(top) # 通过字段匹配
}
```

# 运算

``` less
@test_01: 300px;

.box_02{
    width: (@test_01 - 20) * 5;
}
```

# 嵌套规则

``` less
.list{
    width: 600px;
    height: 400px;

    // 这里的 .li 相当于 .list .li
    .li{
        width: 200px;
    }

    a{
        // & 代表上一层选择器,所以这里是 a:hover
        &:hover{
            color: red;
        }
    }
}
```

# @arguments 变量

``` less
.border_arg(@w:30px,@c:red,@xx:solid){
    border:@arguments;
}

.test_arguments{
    .border_arg(40px);
}
```

# 避免编译

计算/滤镜避免编译 使用 ~'避免编译的代码'

``` less
.test_03{
    width: ~'calc(300px - 30px)';
}
```
