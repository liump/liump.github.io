高德地图开发基础

# 注册高德地图KEY

要使用高德地图首先需要在高德开放平台注册KEY,登录后可以通过右上角控制台-->应用管理中找到.

# Vue 配置

在 Vue 的 build 中的 webpack.base.conf.js(基于vue2.x) 中添加配置: 

externals: {
    'AMap': 'AMap'
},

在需要使用的组件中引入地图对象

import AMap from 'AMap';

在组件的 data 中初始化 amap: null 指定用户渲染的容器

<div id='amap'></div>

::: tip 地图容器需要给定宽高,否则不显示 ::: methods 中初始化方法

handleAMapInit() {
	/*
		new AMap.Map 地图初始化方法
		@params { String } id 地图容器 id
	*/	
	this.amap = new AMap.Map("amap", {
		resizeEnable: true,
		expandZoomRange: true,
		zoom: 20,
		// zooms 拓展地图放大的级别
		// pc max:20 - app max:19
		zooms: [3, 20] 
	});
},

::: tip mounted 中调用,不要在 created中调用 如果使用了 keep-alive 在 activated 中调用 :::

# 设置地图位置及缩放级别

地图位置/缩放的设置有2个方式可以实现:

# 1. 地图初始化的时候使用center属性指定经纬度,zoom属性指定缩放级别

this.amap = new AMap.Map("amap", {
	resizeEnable: true,
	expandZoomRange: true,
	// 指定缩放级别, PC默认最大 18 
	// 如果需要 19 或者 20这2个级别需要上下2个属性 expandZoomRange 和 zooms
	zoom: 20, 
	zooms: [3, 20],
	// 初始化指定坐标位置,如果没有该属性自动寻找所在地的中心坐标
	center: [114.399629, 30.456154] 
});

# 2. 调用API实现

this.amap.setCenter(lnglat);
lnglat即为经纬度数组形如 [114.399629, 30.456154]
this.amap.setZoom(18);
参数为缩放级别,如果初始化时设置了PC最大缩放20,可以传入20,否则大于18均显示18
缩放级别为20的时候,地图2厘米等于实际10米

# 调用地图工具类

高德地图官方自带了很多插件工具 官方定位示例

# 调用方式

handleAMapPlugin() {
    // 调用地图工具库
    AMap.plugin(["AMap.ToolBar", "AMap.Scale"], () => {
    // 调用地图工具,缩放标尺,位移圆钮
    this.amap.addControl(new AMap.ToolBar());
    this.amap.addControl(new AMap.Scale());
    });
},
