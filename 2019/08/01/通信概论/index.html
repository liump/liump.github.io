# 通信的基本概念

通信就是信息的传递和交换;消息是信息的物理表现形式,如语音、文字、图片和数据等；信息是消息的内涵；信号是能够反映或表示信息的传输载体。信源是消息的发源地，其作用是通过传感器吧消息转换为原始电信号。信源可以分为模拟信源和数字信源。信宿是传送消息的目的地。其功能与信源相反，即将复原的原始电信号还原成相应的消息

# 通信方式

通信方式是指通信的双方（或多方）之间的工作方式。根据消息传递的方向与实践关系，可分为单工、半双工、及全双工通信3种。单工通信指消息只能单方向传输的工作方式。例如，广播、遥测、遥控、无线寻呼等都是单工通信方式。

# 通信系统的分类

通信系统的分类：

  1. 按照通信业务分：电报通信系统、电话通信系统、数据通信系统、图像通信系统、综合业务数字通信网。
  2. 按照调试方式为：基带系统和调制（带通）系统。
  3. 按照信号特征分：模拟通信系统、数字通信系统。
  4. 按照传输媒介分：有线通信系统和无线通信系统。
  5. 按照工作波段分：长波通信30~300 kHz，中波通信300kHz ~ 3MHz，短波通信3 ~ 30 MHz，微波通信300MHz ~ 300GHz,光通信43 ~ 3000THz。

# 无线信道

根据通信距离、频率和位置的不同，电磁波的传播方式主要有3种：地（面）波传播 -- 2 MHz以下，传播距离几十到几千米，例如，调幅广播；天波传播 -- 2 ~ 30 MHz 4000km电离层反射方式传播；视线传播 -- 高于 30MHz视线传播距离和收发天线的高度有关。

# 噪声分类

噪声是一种不需要的、有害的电信号。它能使模拟信号失真，数字信号发生错码，限制信息的传输速率。根据噪声的来源分类：

  1. 人为噪声：有人类的活动产生，如电钻和电气开关瞬间造成的电火花；
  2. 自然噪声：自然界中存在的各种电磁波辐射，如闪电，大气噪声；
  3. 内部噪声：通信设备中的电子元器件传输线等产生的噪声、如热噪声、闪弹噪声等。根据噪声的性质分类：

单频噪声 -- 是一种频率单一或频谱很窄的连续波干扰。 脉冲噪声 -- 是一种在时间上无规则的突发脉冲波形。 起伏噪声 -- 是遍布在时域和频域内的随机噪声。如热噪声、闪弹噪声、宇宙噪声。
