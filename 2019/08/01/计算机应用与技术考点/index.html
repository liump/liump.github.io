# 1.计算机的发展过程、计算机的分类和特点

计算机的起源:
法国物理学家帕斯卡与1642年发明了第一台能进行加减法运算的齿轮式加减法器
1673年德国数学家莱布尼兹改进了帕斯卡的设计,增加了乘除法器,制成了可以进行四则运算的机械式计算器.
19世纪20年代,英国数学家查尔斯·巴比奇最先提出了通用数字计算机的基本设计思想,并与1822年和1834年先后设计了差分机和分析机
1936年美国科学家霍华德·艾肯采用机电方法来实现巴比奇分析机的想法,并于1944年研制成功了Markl计算机
冯·若依曼与1946年首先提出了在计算机中存储程序的思想,并确定了存储程序计算机的基本组成和基本工作方法.这一设计思想被誉为计算机发展史上的里程碑.
世界上第一台电子计算机ENIAC在美国与1946年2月14日诞生.这台计算机是在美国陆军军部的主持下,由美国宾夕法利亚大学的埃克特和莫奇里研制成功的.

# 1.1 计算机的发展

  1. 第一代电子管计算机(1946-1957年):
    体积庞大、运算低(每秒几千次到几万次) 、成本高、可靠性差、内存容量小,主要用于科学计算、军事和科学研究,比如ENIAC、IBM650(小型机) 、IBM709(大型机)等。
  2. 第二代晶体管计算机(1958-1964年):
    体积相对较小,运算速度已提高到每秒几十万次,可靠性和内存也有较大提高,数据处理和自动控制,汇编语言与高级语言,除用于军事和科学研究工作外还用于数据处理和事务处理,比如IBM7090 、IBM7094和 CDC7600等。
  3. 第三代集成电路计算机(1965-1970年):
    体积、耗电量相对进一步减少,运算(每秒几十万次到几百万次)速度和可靠性进一步提高,使用了操作系统,外部设备种类繁多,应用更加广泛,比如IBM360系列 、富士通F230系列等。
  4. 第四代大规模和超大规模集成电路计算机(1971年至今):
    体积小、耗电量大大减少,运算速度可以达到每秒上千次到万亿次,计算机的存储容量和可靠性又有了很大提高，功能更加完备，应用进入网络时代。（办公室、学校和家庭）

# 1.2 计算机的分类

  1. 按计算机的用途分类
    科学与工程计算计算机: 用于科学与工程领域的计算问题
    数据处理计算机: 用于数据处理,如:报表统计
    工业控制计算机: 用于生产过程监测和控制
  2. 按计算机的使用范围分类
    专用计算机: 某种特定问题而设计的计算机,有运算速度快/精度高/运行效率好/针对性强和结构简单等特点.如银行存取款/飞机自动控制/数控机床等
    通用计算机: 解决各种一般问题而设计的计算机,有很强的综合处理能力
  3. 按计算机的规模和处理能力分类(指计算机的大小/存储容量/运算速度/字长/外部设备等多方面的综合性能指标)

# 2.计算机应用领域和计算机发展趋势

计算机作为一种信息处理工具，它具有极高的处理速度、很强的存储能力、精准的计算和逻辑判断能力以及自动控制能力，具有运算速度快、计算精度高、记忆力强、可靠性高和通用性好的特点，因此被广泛用于各个领域，归纳可分为以下几个方面：

  • 科学计算（人造卫星轨迹的计算，房屋抗震强度的计算，火箭及宇宙飞船研究等）数值计算
  • 数据处理（人事管理、库存管理、财务管理、图书资料管理、图像处理等） 非数值形式
  • 自动控制（钢铁企业、石油化工业、医药工业等生产中）
  • 计算机辅助系统
    • CAD(Computer Aided Design 计算机辅助设计)：飞机、船舶、建筑设计等
    • CAM(Computer Aided Manufacturing 计算机辅助制造)：计算机监视系统等
    • CAT(Computer Aided Test 计算机辅助测试)：利用计算机进行复杂大量的测试工作CAI(Computer Aided Instruction 计算机辅助教学)：个别教学等。
  • 人工智能（Artificial Intelligence,AI）：机器人。
  • 多媒体计算应用 多媒体（Multimedia）----文本、音频、视频、动画、图形和图像等各种媒体综合起来。

随着网络技术的发展，计算机的应用进一步深入到社会的各行各业，电子邮件、电视电话、电视会议、文档传输、电子教育、电子娱乐、电子购物、远程医疗和会诊、交通信息管理等。

# 信息的表示

# 3.1 数制 数制即表示数值的方法,有进位计数制和非进位计数制两种

  1. 进位计数制: 按照进位的原则进行计数的数制称为进位计数制,简称进制.
  • 例如:
    • 日常生活十进制
    • 二十四进制(24小时为一天)
    • 六十进制(60分为1小时,60秒为1分)
    • 二进制(鞋/袜子等两只为一双)
    • 特点:
      • 使用固定个数的数码表示数值的大小.数码的个数 R 称为该数制的基数 (Radix),最小数码是0,最大数码是 R-1.数码在一个数中所处的位置称为数位.
        • 例如:
        • 十进制数(Decimal)的基数是10,使用0~9十个数码;
        • 二进制数(Binary)的基数是2,使用0~1两个数码;
        • 八进制数(Octal)的基数是8,使用0~7八个数码;
        • 十六进制数(Hexdecimal)的基数是16,使用0~9 a~f(A~F)十六个数码;
      • 逢 R 进一.在各种进制中,有一套统一的规则.R进制的规则是逢R进1.
        • 例如:
        • 十进制数逢10进1
        • 八进制数逢8进1
        • 二进制数逢2进1
        • 十六进制数逢16进1
      • 采用位权表示法.表示数值大小的数码与它在数中的位置有关,即与它所在位置的"权"值有关.位权值得大小是以基数R为底,以数码所在位置的序号为指数的整数次幂.
        • 例如:
        • 十进制整数部分的位权值从个位开始向左依次为10的0次方, 10的1次方, 10的2次方..., 十进制小数部分的位权值从小数点后第一位开始向右依次为10的-1次方, 10的-2次方...
  1. 非进位计数制: 表示数值大小的数码与它在数中的位置无关的数制称为非进位计数制.
    • 例如: 罗马数字就是典型的非进位计数制
