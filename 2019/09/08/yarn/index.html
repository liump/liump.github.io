yarn 命令

# yarn add

yarn add [package]@[version]
这将安装您的dependencies中的一个或多个包。
用 --dev 或 -D 会在 devDependencies 里安装一个或多个包。

# yarn cache

yarn cache dir yarn 的全局缓存位置

yarn cache clean
运行此命令将清除全局缓存。 将在下次运行 yarn 或 yarn install 时重新填充。

# yarn list

显示所有包和依赖

# yarn remove

yarn remove <package...> 移除指定包

# yarn upgrade

yarn upgrade 升级所有依赖包,也可以后面接包名用于升级指定的包

# yarn global bin

yarn global bin 查看全局安装的根目录

# yarn指定淘宝源 :

yarn config set registry http://registry.npm.taobao.org
