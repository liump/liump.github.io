# 说明

根据项目需求,PC端使用 vue-baidu-map 二次开发.地图相关代码都放置在 gitblit 公司仓库 zjtt_pc/src/bmap.

# 使用

通过 elementUI 的 drawer(抽屉) 的方式展示大屏地图.

参数

  1. parentId 子项目ID,通过项目ID保存地图数据.
  2. model 使用驻地网兼容,后期可以删除兼容相关代码.
  3. tableType 有 1 | 2 可选,当传递为1(预算绘图),当传递为2(现场绘图)
// 在 vue 文件中使用
<el-drawer
  title="现场绘图"
  :visible.sync="showBmap"
  size="80%"
  append-to-body
  destroy-on-close
>
  <bmap v-if="showBmap" :parentId="parentId" model="zdw" :tableType="tableType" ref="bmap"></bmap>
</el-drawer>

# 功能说明

  1. 点信息变更: 修改节点对应的数据
  2. 线段工作内容: 修改线段对应的数据
  3. 光缆列表: 用于收集数据,后端生成CAD.
  4. 反馈: 刘总提出施工队不可以删除节点,只能文员删除,于是此需求应运而生.当施工队在反馈弹框中输入修改需求,文员通过说明修改后通知施工队.并纪律修改流程.
  5. 线图及补充功能: 通过黄总生成的JSON数据在地图上生成地形轮廓图,便于施工队寻找参照物提供帮助.
